# pcsx2_europe_memcard_collection
EU based PCSX2 .ps2 virtual Memcard Collection with EU based Savegames

## Overview
PCSX2 uses virtual PlayStation 2 memory cards stored as `.ps2` files.  
These files behave like original PS2 memory cards and store game save data, system configuration, and profile-related data.

## What `.ps2` Memory Cards Are
- Virtual PS2 memory cards used by PCSX2
- Usually assigned to **Slot 1** and **Slot 2**
- Stored as standalone `.ps2` files on your system
- Can be created, renamed, backed up, replaced, and archived manually

## Main Features
- Separate save storage per virtual card
- Works like original physical PS2 memory cards
- Supports multiple cards for different setups or game collections
- Easy backup by copying the `.ps2` file
- Useful for separating regions, genres, users, or long-term playthroughs
- Helps keep save data organized and reduce card clutter

## Typical Use Cases
- One card for all general saves
- Separate cards for RPGs, racing games, or region-specific games
- Dedicated cards per user
- Backup/archive cards before testing cheats, patches, or save imports
- Clean cards for troubleshooting save-related issues

## Basic Usage in PCSX2
1. Open **PCSX2**
2. Go to the memory card settings / manager
3. Create a new virtual memory card if needed
4. Assign the desired `.ps2` file to **Slot 1** or **Slot 2**
5. Start the game and save normally in-game

## MyMC / Memory Card Editor Note
For direct management of `.ps2` memory card files on PC, you can use **MyMC / mymc**.  
It is useful for importing, exporting, and inspecting PS2 save data stored inside virtual memory card files.  
Before editing or importing saves, always create a backup of the original `.ps2` file first.

## Recommended Organization
- Use clear file names  
  Example: `Main-Slot1.ps2`, `RPG-Collection.ps2`, `EU-Saves.ps2`
- Keep regular backups
- Store archived cards in a separate backup folder
- Track which saves are stored on which card

## Backup Recommendation
To back up a memory card, simply copy the related `.ps2` file to another folder, drive, or cloud backup location.

## Notes
- A `.ps2` file may contain multiple game saves
- Some games also create system or settings data
- Keep card naming consistent to avoid confusion
- Before editing, importing, or replacing saves, create a backup copy first
